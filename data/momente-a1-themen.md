# Карта тем Momente A1 (24 урока · 8 модулей)

> Источник: **Momente A1 Kursbuch (Hueber, 2021), оглавление (Inhalt), стр. 6–9** — извлечено из materials/. Надёжно (сам учебник).
> Колонка «der-категории» = какие группы мужского рода даёт тема для сериала про Otto. Пометка «сверить» = слова ещё НЕ извлечены из Bildlexikon урока, не выдуманы.
> Тип блока: 🖼 сцена (богатый набор предметов der → картинка) · 📖 сюжетный (двигает историю, объектов мало).

## Итог по типам
- 🖼 сцены-картинки (богатый der): L3, L4–L6, L8, L9, L10, L12, L13, L15, L16, L18, L22, L23 — ~14 блоков.
- 📖 сюжетные (двигают историю): L1, L2, L7, L11, L14, L17, L19, L20, L21, L24 — ~10 блоков.
- **Вывод:** der тянется через весь А1 (целые категории — дни/месяцы/сезоны/погода/мужская родня/еда/тело/одежда), поэтому сериал про Otto живой во всех модулях. Образ из L1 (килт+усы+ключи) окупается в L22 (Одежда).
