Глаголы

Каркас раздела. У каждого глагола: формы по лицам (ich/du/er…) + прошедшее (Perfekt). Карточки-группы со своей сценой и текстом + аудио.

Модальные v3

können, müssen, wollen, sollen, dürfen, möchte.

Прошедшее простое v3

Правильные/слабые — простая форма Partizip II.

Прошедшее сложное v3

Сильные/неправильные — по группам.

Алфавитный глоссарий v3

Все глаголы списком: клик → формы + прошедшее. Для поиска и печати.
Старт — L19 «Im Haushalt» (глаголы быта), чтобы стыковалось с темой «Быт».
