# 06. Правила написания сквозного рассказа

> Этот документ — спецификация того, КАК пишутся тексты-истории. Можно дорабатывать отдельно.

## Суть
Не набор проходных текстов, а **единый сквозной рассказ на весь уровень А1** — как сериал. Один большой текст про персонажа, разрезанный на части по темам, по порядку. Каждая тема = очередная «серия». К концу А1 — «финал сезона»: ученик целиком представляет персонажа.

## Принципы
1. **Сериал, не эпизоды.** Истории связаны, тянутся одной нитью от темы к теме. Можно «вырвать» кусок под конкретную тему — он самодостаточен, но является частью целого.
2. **Персонаж раскрывается прогрессивно.** С каждой темой узнаём о нём больше: характер, профессия, хобби, привычки, любимые вещи. Он должен быть как герой сказки/мультика — запоминающийся, с чертами.
3. **Всё, чем наделяем героя, — строго его рода.** Профессия, хобби, любимые предметы, привычки — из слов рода персонажа. Это и есть мнемоническая нить: вспомнил вещь → вспомнил героя → вспомнил род.
4. **Характер делает слова «липкими».** Не «сегодня он писал письмо», а «он замкнутый, недосыпает, каждый день пьёт кофе, всюду гремит связкой ключей, любит писать длинные письма». Привычка + эмоция + повтор = запоминается.
5. **Глаголы — только из пройденных** (или чуть-чуть забегая вперёд, помечая). Уровень строго А1.
6. **Story bible.** Ведём свод фактов о каждом герое, чтобы не было противоречий (особенно по родам предметов) между сериями.
7. **Три параллельных линии:** Otto (der), Грета (die), Тео/дети (das). Каждая — своя нить через все темы.

## Пример героя der — Otto (черновик образа)
- Характер: хмурый, замкнутый, вечно невыспавшийся.
- Привычки: каждый день пьёт **der Kaffee**; всюду носит и гремит связкой **der Schlüssel**; любит писать **der Brief**.
- Хобби/сфера (всё мужского рода): **der Garten** (возится в саду), **der Sport / der Fußball**.
- Профессия: подобрать из слов мужского рода (напр. связано с офисом/техникой — der Computer, der Drucker).
- НЕЛЬЗЯ: газета (die Zeitung — ж), куртка (die Jacke — ж), очки (die Brille — ж), велосипед/машина (das Fahrrad/das Auto — ср).

## Формат текста под тему
- Короткий связный рассказ А1, вплетающий слова рода этой темы.
- Под картинкой(ами) сцены.
- Озвучивается (аудио).
- Целевые слова можно скрывать «глазиком» (самопроверка).
