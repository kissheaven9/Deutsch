# 05. Видение продукта и роадмап

## Главный принцип: память-дворец, не зубрёжка
Цель — не запомнить за один присест, а **хранилище-словарь + ассоциативный вызов**: вспомнил ОДНУ сцену-историю → всплыло много слов. Поэтому сцена богатая и сюжетная; «7±2 за подход» здесь вторично — это справочник, к нему возвращаются. Сколько кадров на сцену — вопрос вёрстки, не учебной порции.

## Сквозной сюжет (story bible) — как сериал
Три параллельные непрерывные истории: Мужчина (der), Женщина (die), Ребёнок/дети (das). Из темы в тему один и тот же герой делает всё больше, раскрывается как персонаж (на уровне А1). Вспоминая вещь → по сюжету доходим до персонажа → его род (нейросвязь). У каждого героя — любимые вещи/привычки из слов ЕГО рода (Отто пьёт der Kaffee, носит связку der Schlüssel, читает der Roman/der Brief). Нужен **свод истории (bible)**, чтобы линии были связны и без противоречий по родам.

## Модули продукта (полный список; не всё в v1)
1. **Существительные**: тема → род (3 персонажа) → сцена + картинка + подписи (слой) + текст + аудио + глазик.
   - Подраздел **«Множественное число»**: по типу окончания (+n/-en, +e/-¨e, +er/-¨er, +s, без изм./умлаут), своя картинка/стих/аудио/глазик.
2. **Глаголы**: модальные / прош. простое / прош. сложное; формы по лицам + Perfekt; карточки-группы со сценой; текст + аудио. **Отдельный раздел** (прош. время L11 и пр. — сюда, не в существительные). Свои топики: один глагол в разных лицах/формах, тоже про Otto. **Синхрон по уровню** с существительными: тексты не используют глаголы будущих уроков/А2.
3. **Словари (накопительные)**: существительные по темам; глаголы — по группам + алфавитный глоссарий.
4. **Топики (graded readers)**: после каждой темы — несколько текстов/историй/сказок, кумулятивно переиспользуют ВСЕ пройденные слова + новые (закрепление через повторные встречи, ~7–8+). Аудио + глазик.
   - **НЕ разделены по роду** (в отличие от сцен). Лежат «в той же плоскости», что выбор рода внутри темы: открыл тему → смотришь род (сцены) ИЛИ читаешь топики к ней.
   - Роль — **закрепление, не знакомство**: слово, уже выученное в сцене со своим родом, встречается снова в связном тексте. К теме 15 топик включает все темы до неё (все слова + пройденные глаголы).
   - Раскрывают Otto дальше (сестра, брат, дядя; куда ходит по дням). Уровень и набор глаголов — **синхронны текущему уроку** (без забегания, без А2).
5. **Экспорт PDF**: словарь по теме; выбор тем → сборка PDF; полный печатный (колонки der/die/das + глаголы отдельно, по темам); кумулятивный (все темы до текущей включительно); только глаголы (алфавит); каждый топик; блок «слова+картинка+текст» (офлайн, не зависеть от сайта).
6. **Аудио везде** (TTS, предгенерация) — для аудиалов.

## Решения по словарям/экспорту
- Существительные — по ТЕМАМ (семантика помогает памяти), не по алфавиту.
- Глаголы — основной вид по группам (метод) + алфавитный глоссарий для справки/печати. Оба дёшевы → делаем оба.

## Коррекции рода (из обсуждения; держим верно)
- газета = die Zeitung (ж) → Грета.
- машина = das Auto, велосипед = das Fahrrad (ср) → Тео.
- ключ = der Schlüssel (м) → Отто (связка ключей — его фишка).
- кофе = der Kaffee, чай = der Tee (м) → Отто.

## Фазы (сначала тонкий срез, чтобы не проектировать бесконечно)
- **v1 (MVP) — ЗАМОРОЖЕНО:** один персонаж Отто (der), одна тема «Быт», полный цикл ОДНОЙ сцены: сюжетный текст А1 + картинка (ChatGPT вручную) + подписи слоем + аудио + глазик по словам. Цель — рабочий цикл вживую.
- **v2:** Грета и Тео по «Быту»; сводная сцена; подраздел Pl.
- **v3:** глаголы (старт L19 Im Haushalt); карточки-группы.
- **v4:** топики (graded readers); сквозной сюжет между темами.
- **v5:** PDF-экспорт во всех вариантах.
