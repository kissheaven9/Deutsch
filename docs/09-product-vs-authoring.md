# 09. Граница: финальный продукт vs авторская кухня (два пространства)

> Зафиксировано 2026-06-04. Чтобы два чата не лезли в работу друг друга.

## Два пространства
1. **Финальный продукт (ЭТОТ чат).** Сайт, который пользователь всегда открывает на локальном сервере (`site/`, порт 8000) и видит, как он меняется. Позже — выкладка на хостинг для педагога.
   - Поток: **Главная → Тема → (род der/die/das) → Сцена** (+ глаголы, другое, топики внутри темы).
   - Это конечный продукт. Его строит и поддерживает этот чат.
2. **Авторская кухня (ДРУГОЙ чат).** Вспомогательные HTML-страницы, где прописываются **тексты-истории под картинки и промпты** для ChatGPT. Рабочее место пользователя, чтобы смотреть тексты для будущих уроков.
   - Не часть продукта, в навигацию продукта не входит.

## Правило границы
- Этот чат **не редактирует** авторские страницы (тексты+промпты, напр. `otto-szene-1.html`, `otto-der-szenen.html`, `otto-arc.html`). Только **читает** готовые тексты/канон оттуда.
- Другой чат не редактирует продуктовые страницы.
- **Общий источник правды** (читают оба, не ломая): `data/story/otto-bible.md` (канон), `data/themen/*` (лексика), `docs/06,08` (правила историй/ТЗ).

## Страницы ПРОДУКТА (этот чат)
- `site/index.html` — главная (список тем).
- `site/thema-01.html` — тема 1 (существительные по роду + глаголы + другое + топики).
- `site/thema-01-der.html` — **сцена der → Otto** (картинка + подписи + слова + текст + упражнение). Шаблон сцены для всех родов/тем.
- далее: `thema-01-die.html` (Грета), `thema-01-das.html` (Тео), сцены других тем, раздел глаголов, словари, экспорт.

## Страницы АВТОРСКОЙ кухни (другой чат, не трогаю)
- `site/otto-szene-1.html` — 3 колонки: слова · текст-история · промпт. Вспомогательная.
- `site/otto-der-szenen.html`, `site/otto-arc.html` — раскладка/роадмап историй.

## Шаблон страницы сцены (продукт) — единый для всех персонажей
1. Большая картинка сцены + **подписи слоем**: имя+родство видно, **значения-слова (профессия и т.п.) под тапом** (самопроверка).
2. **Слова сцены** (по роду) с переводом: 🔊 послушать, 👁 скрыть/открыть перевод (все/по одному).
3. **Текст** (немецкий + русский) с аудio (играть/пауза/сначала).
4. **Упражнение**: вопросы про персонажей (перевод по кнопке; поле ввода + «Проверить» 🟢/🔴 + «Показать ответ»).
