Множественное число

Те же существительные, но сгруппированы по типу окончания. Под каждый столбик — своя картинка, стишок и аудио (как у рода). Каркас структуры:

+ (e)n

  • die Lampe → Lampen
  • die Nachricht → Nachrichten
  • die Tastatur → Tastaturen

+ e / ¨e

  • der Tisch → Tische
  • der Stuhl → Stühle
  • der Schrank → Schränke

+ er / ¨er

  • das Bild → Bilder
  • das Passwort → Passwörter

+ s

  • der Laptop → Laptops
  • das Sofa → Sofas
  • das Tablet → Tablets

– / ¨ (без оконч.)

  • der Drucker → Drucker
  • der Spiegel → Spiegel
  • der Sessel → Sessel
Формы мн.ч. — черновик, сверяем с Lernwortschatz (Arbeitsbuch), не выдумываем. Под каждый столбик добавим картинку + стих + аудио + «глазик» на формах.
