# Deutsch — система запоминания немецкой лексики A1

Сайт для запоминания немецких слов: род привязан к персонажу (der→Otto, die→Грета, das→Тео/Lina), лексика подаётся через сюжетные тексты-истории, картинки и мультисенсорику (читать / слушать / проговаривать). Глаголы — по методике «семья = персонаж» + лестница из 8 шагов. Лексика **возвращается из темы в тему** (спираль), чтобы не забывалась.

Сейчас — **под себя и на тесте** (пользователь + репетитор). Перспектива — открыть для других.

## Где живёт
- **Живой сайт:** https://kissheaven9.github.io/Deutsch/ (GitHub Pages).
- Сайт **статический** — обычные HTML/CSS/JS, без сервера и сборки.

## Как запустить
- **Локально:** открыть `site/index.html` в браузере. Для корректной работы относительных путей лучше через локальный сервер:
  ```bash
  cd site && python3 -m http.server 8000   # затем открыть http://localhost:8000
  ```
- **Обновить живой сайт (деплой):**
  ```bash
  git push origin main
  git subtree push --prefix=site origin gh-pages
  ```
  Ссылку после деплоя давать с анти-кэшем (`?v=<число>`), при сомнениях — открывать в инкогнито.

## Источник лексики
Курс строится по учебнику **Momente A1** (Kursbuch + Arbeitsbuch, лежат в `materials/`): **Тема N = Lektion N**. Берём оттуда **список слов и глаголов по уровню** (Lernwortschatz + упражнения урока); ориентир уровня сверяем с открытым стандартом **Goethe A1 Wortliste**. **Тексты и упражнения пишем свои** (оригинальные истории про Otto/Грету/Тео) — учебник даёт «какие слова на уровне», а не готовый контент.
> `materials/` (учебники) — **локально, НЕ публиковать** (копирайт; в `.gitignore`).

## Статус
В разработке, не «сбор требований». Уже собрано: темы 1, 2, 3 (род по персонажам, сцены, словари, упражнения), тема 20 (глаголы), тренажёры глаголов, методика «семья=персонаж» + лестница, принцип спирального повтора. **MVP — тема «Быт»** (household, L4+L6+L15), полный цикл как образец.

## Структура репозитория
- `site/` — сам сайт (страницы тем/сцен, `js/`, `css/`, `img/`, `audio/`). Деплоится в ветку `gh-pages`.
- `docs/` — документация и правила (см. карту ниже).
- `data/` — данные: словари тем, story-bible персонажей.
- `materials/` — учебники Momente (**не публиковать**).
- `tools/` — вспомогательные скрипты (генерация аудио, данных).
- `Дизайн/` — дизайн-система и живой кит.
