# Deutsch — система запоминания немецкой лексики А1

Сайт для запоминания немецких слов через привязку рода к персонажу, сюжетные истории, визуализацию и мультисенсорику (читать / слушать / проговаривать). Сейчас — под себя, с прицелом на будущую монетизацию.
